# NestJS + gRPC: a multi microservices example

This repo show as to configure a multi microservices stack in NestJS using gRPC.

## Stack
* 2 microservices 
* 1 client

## Run it!
```bash
make npm_install
docker-compose up
```
Navigate to

```
http://localhost:3000/test/client1
http://localhost:3000/test/client2
```
